## Description du projet

Ce projet consiste à concevoir et développer une boutique en ligne moderne
à l’aide de Next.js.

Le site permettra :

- d’afficher les produits
- de consulter les détails d’un produit
- de naviguer entre les sections
- d’offrir une interface simple et responsive

## Technologies utilisées

- Next.js
- React
- TypeScript
- CSS

## Installation

Cloner le projet :

```bash
git clone https://github.com/saintlouispierrerichard6-prog/cours_site_web_avance_Projet_creation_boutique_en_ligne.git
```

Installer les dépendances :

```bash
npm install
```

Démarrer :

```bash
npm run dev
```
